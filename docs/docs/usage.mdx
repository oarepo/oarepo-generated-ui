# Usage

To render a React UI from a layout definition, you need to provide
the definition and (optional) data context to provided `GeneratedUI` React component:

```jsx live
<GeneratedLayout
  config={[
    {
      component: 'label',
      color: 'red',
      children: 'This label was generated from config!',
    },
  ]}
></GeneratedLayout>
```
